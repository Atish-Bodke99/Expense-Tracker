# 💸 Expense Tracker (React App)

A simple and user-friendly Expense Tracker application built using React. It allows users to track their income and expenses, view balance, and manage transaction history in real-time.

---

## 🚀 Features

* 💰 Displays current balance
* 📈 Tracks total income and expenses
* ➕ Add new transactions (income or expense)
* 📜 Transaction history with details
* 🟢 Income and 🔴 Expense indicators
* 📱 Clean and responsive UI

---

## 🖼️ UI Overview

* Shows **Total Balance**
* Displays **Income vs Expense summary**
* Maintains **Transaction History**
* Simple form to add new transactions

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* CSS3
* React Hooks (useState)

---

## 📌 How It Works

* Enter transaction text and amount
* Positive amount → Income
* Negative amount → Expense
* Balance updates automatically
